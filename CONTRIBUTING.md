# Contributing

Thanks for improving OpenMeet.

## Development

```bash
npm install
npm run typecheck
npm run build
```

## Before Opening a Pull Request

- Do not commit `.env`, SQLite databases, OAuth client JSON files, private keys, or downloaded credentials.
- Run `npm audit --audit-level=moderate`.
- Keep configuration examples generic and use `example.com` addresses for placeholders.
- Report security issues privately; do not include exploit details or secrets in public issues.
